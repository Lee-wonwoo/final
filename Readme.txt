 이 작업의 목적은 1992년과 1994년 사이에 Laboratories Cambridge에서 찍은 얼굴 이미지를 가지고 조금씩 다른 이미지를 같은 사람으로 판별할 수 있는지 알고리즘을 만드는 것입니다. 
 제가 선택한 알고리즘은 LogisticRegression(로지스틱 회귀) 알고리즘입니다. 선형 회귀 방식을 분류에 적용한 알고리즘이며 주로 $p = {1 \over 1+e^{−z}}$ 형태의 시그모이드(Sigmoid) 함수 최적선을 찾아 이 시그모이드 함수의 반환 값을 확률로 간주하여 분류를 진행합니다.
 로지스틱 회귀의 알고리즘에는 penalty(규제종류) class_weight(가중치) 등의 파라미터가 있으며 제가 조절한 파라미터는 C와 max_iter입니다. max_iter는 계산할 작업수이며 C는 규제 강도의 역수입니다 C값이 클 수록 규제 강도가 적어집니다.
C와 max_iter를 너무 작거나 크게 하면 정확도가 떨어집니다.
